## 🎮 Retro Game Vault

### 📌 Project Overview

Retro Game Vault is a web-based application that allows users to search and explore both classic and modern video games. It provides an intuitive interface to discover games, view detailed information, and manage a personalized wishlist — all powered by a public gaming API.

The main goal of this project is to build a fast, responsive, and visually engaging game search engine while applying real-world frontend concepts such as API integration, state management, and performance optimization.

---

### 🚀 Purpose

This project is designed to:

* Help users discover video games across different platforms
* Provide detailed information such as ratings, genres, and release dates
* Allow users to save their favorite games locally (without requiring login)
* Practice modern web development skills using APIs and dynamic UI

---

### 🌐 API Used

**RAWG Video Games Database API**
👉 [https://rawg.io/apidocs](https://rawg.io/apidocs)

The API provides:

* Game search functionality
* Detailed game information (ratings, platforms, release dates, etc.)
* Images and metadata for thousands of games

### ✨ Features

#### 🔍 Game Search (with Debounce)

* Real-time game search
* Debounce logic to reduce unnecessary API calls
* Improves performance and user experience

#### 🎮 Platform Filtering

Filter games by platform:

* PC
* PlayStation
* Xbox
* Nintendo

Makes it easier to narrow down results.

#### ⭐ Favorites / Wishlist

* Add games to a wishlist
* Stored in browser using `localStorage`
* Data persists even after page refresh

#### 📊 Compare Mode (Challenge Feature)

* Select two games
* View details side-by-side

Compare:

* Ratings
* Release dates
* Platforms
* Genres
